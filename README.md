# AI-Based Structural Analysis of Ag–NHC Complexes
This repository contains the dataset and R scripts used for unsupervised machine learning analysis of bond length data in ligands and Ag–NHC complexes.
## Methods
- Principal Component Analysis (PCA)
- Heatmap visualization
- Clustering analysis
- Statistical comparison of bond features
## Files
- ligand-complex-raw-data.xlsx (dataset)
- analysis_code.R (analysis script)
## Purpose
This work supports a research study on data-driven structural analysis of metal–ligand systems.
